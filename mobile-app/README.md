# PDP Mobile Prototype

Expo Router prototype implementing the requested backlog items:

- `SCRUM-29` Farmer Login and Session
- `SCRUM-30` Create and Edit Farm Profile
- `SCRUM-36` Log Key Compliance Actions
- `SCRUM-39` Duplicate Previous Report as Draft

Included dependency surfaces:

- personalized task list to reflect profile rules
- basic report submission and report history to support draft duplication

SDK baseline:

- Expo SDK 54
- React Native 0.81
- React 19.1

## Run

1. `npm install`
2. `npm run start`
3. open on Android with Expo Go
